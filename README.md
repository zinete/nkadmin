# 基于 next + koa2 的管理后台

自己写一些小项目用的管理后台。完善中

## Deployment

git clone 之后执行以下命令

```bash
  yarn install
  yarn run dev
```

## Running Tests

To run tests, run the following command

```bash
  yarn run test
```

## Environment Variables

根目录修改下面 2 个环境变量。可配置启动的端口和域名

`BASE_HOST`

`BASE_PORT`
